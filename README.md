# Kata Web 4 - Formulaire HTML

## Description

Ce projet a été réalisé dans le cadre d'un exercice d'entraînement HTML.

L'objectif est de créer un formulaire contenant :

- un champ texte pour le nom.
- un champ email.
- une liste déroulante.
- une zone de texte pour le message.
- des champs obligatoires.
- un bouton d'envoi.

## Technologies utilisées

HTML5.
